# Private Server Gateway (Roblox → Discord)

Минималистичная “премиум” страница-переходник для входа на приватный сервер Roblox.
Используется как ссылка для кнопки/Embed в Discord: пользователь открывает страницу в браузере и заходит в игру через Roblox Player или по ручной инструкции.

## Что делает
- **Открывает Roblox Player** по deep-link (`roblox://...`) кнопкой **«Открыть в Roblox»**
- Даёт **ручной способ входа** (покупка игры → вступление в группу → Private Servers → ввод Join Code)
- Внизу карточки — **превью-постер** с кнопкой **Discord** (ведёт на приглашение сервера)

## Файлы
Обязательные:
- `index.html`

Опциональные (если используешь изображения):
- `banner.jpg` — фон страницы (если нет, будет красивый градиент)
- `preview.jpg` — картинка внизу левой карточки
- `steps-collage.webp` (или `.jpg/.png`) — коллаж со “шпаргалкой” шагов в правой панели

> Все файлы можно держать в корне рядом с `index.html` (как сейчас).  
> Если переносишь в папку `assets/`, не забудь обновить пути в `index.html`.

## Настройка
Все основные параметры меняются внизу `index.html` в секции `// === Settings ===`:
- `PLACE_ID` — ID игры
- `JOIN_CODE` — код приватного сервера для ссылки
- `MANUAL_JOIN_CODE_TEXT` — код для ручного входа (например `CALYX`)

Ссылки:
- Discord invite — в блоке кнопки **Discord** внутри `mediaBtn`
- Roblox group — в правой панели кнопка **«Открыть группу»**

## Деплой (GitHub Pages)
1. Репозиторий → **Settings → Pages**
2. **Deploy from a branch**
3. Branch: `main`, folder: `/ (root)`
4. Сохрани — сайт будет доступен по ссылке вида `https://<username>.github.io/<repo>/`

## Использование в Discord
В кнопку/Embed ставь URL GitHub Pages страницы (HTTPS).  
Discord не поддерживает прямые `roblox://` ссылки, поэтому используется этот gateway.
